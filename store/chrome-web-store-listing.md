# Chrome Web Store 上架文案

## 扩展名称
Chrome 快捷键速查

## 一句话卖点
长按一个键，立刻弹出 Chrome 常用快捷键速查表。

## 简短描述
长按自定义键显示 Chrome 常用快捷键，自动识别 Mac / Windows，松开即关闭。

## 详细描述
Chrome 快捷键速查把常用快捷键放到你正在浏览的网页里。

按住你设定的触发键一小段时间，扩展会在页面右上角显示速查面板；松开按键，面板立刻关闭。整个过程很轻，不打断当前浏览流程，适合刚开始记快捷键、偶尔会忘快捷键、或跨 Mac / Windows 切换使用 Chrome 的人。

功能特点：

- 长按触发，松开关闭
- 自动识别 Mac / Windows 快捷键布局
- 支持在选项页修改触发键
- 支持自定义长按时长
- 使用本地存储保存设置
- 无账号、无登录、无远程服务

适合这些场景：

- 想快速查 Chrome 常用快捷键
- 从 Windows 切到 Mac，或从 Mac 切到 Windows
- 记不住开发者工具、标签页、地址栏、书签等操作快捷键
- 想减少反复搜索“Chrome shortcut”的次数
